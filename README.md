# 🎂 Birthday Freebies Tracker

紀錄美國灣區（以 SJSU 為中心）各品牌的生日優惠，包含領取條件、時效與 CP 值評比。

## 內容

| 欄位 | 說明 |
|------|------|
| 店名 | 品牌名稱與補充備註 |
| 類別 | 餐廳/速食、飲料/咖啡、甜點/冰品、美妝/零售 |
| 贈送物品 | 免費品項說明 |
| 需會員？ | 是否需要 App / 帳號，以及加入條件 |
| 領取期限 | 有效時間窗口 |
| CP 值 | 高 / 中 / 低 |
| 距 SJSU | 近 / 中 / 遠 |

## 使用方式

用瀏覽器開啟 `index.html`，或在支援 HTML 的平台（如 Notion、GitHub Pages）嵌入使用。可透過頂部篩選按鈕依類別、CP 值或領取批次過濾。

目前收錄約 **30 間** 品牌，涵蓋 Starbucks、Sephora、Chipotle、Cheesecake Factory、Dutch Bros 等。

## 資料來源

各品牌官方 Rewards 計畫條款（2026 年 3 月整理）。
